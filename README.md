# Music platform backend

This repository contains code of a learning project called: "Music platform".

It's purpose is to build a REST API that allows users to find songs/creators.

## Learning

- Spring Boot
- Spring Data JPA
- Spring Security
- Spring testing
- MeiliSearch
- Weaviate
- Redis
- Caching
- Rate limiting
- UML diagrams
- Patterns
- SOLID
- GRASP
